Overview

El Cochinito is a comprehensive iOS application designed to help parents teach their children financial responsibility through virtual money jars and automated allowance tracking. The app (Spanish for "piggy bank") makes it fun and easy to manage allowances, track savings goals, and teach valuable money management skills.

Technical Stack

  • Language: Swift
  • UI Framework: UIKit with Storyboards
  • Storage: JSON file-based persistence with iCloud Documents integration
  • Design Pattern: MVC (Model-View-Controller)
  • Cloud Services: iCloud Document Storage for seamless device synchronization
  • Data Management: Core Data for local persistence

Architecture Highlights

SimpleStorageManager

A singleton class handling all data operations with JSON-based file storage, automatic migration to iCloud, and NSMetadataQuery for monitoring file changes.
